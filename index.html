¿Qué son las Sentencias de Control en Java?

Las sentencias de control en Java permiten decidir qué bloques de código ejecutar dependiendo de ciertas condiciones. Estas estructuras son esenciales en cualquier lenguaje de programación y permiten controlar el flujo de un programa.

Java incluye sentencias de control condicional como if, else y switch, así como estructuras repetitivas como for, while y do-while. Gracias a estas herramientas, los programadores pueden crear algoritmos complejos, responder a condiciones en tiempo real y automatizar tareas repetitivas de forma eficiente.

También existen sentencias de salto como break, continue y return, que permiten alterar la ejecución del flujo desde dentro de ciclos o métodos.

Sentencia If / Else

Permite ejecutar código condicionalmente dependiendo de si una expresión es verdadera o falsa.

if (edad >= 18) {
  System.out.println("Eres mayor de edad");
} else {
  System.out.println("Eres menor de edad");
}

Sentencia Switch

Permite seleccionar entre múltiples opciones según el valor de una variable.

switch(dia) {
  case 1:
    System.out.println("Lunes");
    break;
  case 2:
    System.out.println("Martes");
    break;
  default:
    System.out.println("Otro día");
}

Bucles: For, While y Do-While

Estructuras que repiten un bloque de código mientras se cumpla una condición.

for (int i = 0; i < 5; i++) {
  System.out.println(i);
}

int i = 0;
while (i < 5) {
  System.out.println(i);
  i++;
}

do {
  System.out.println(i);
  i++;
} while (i < 5);
